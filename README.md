###  Run
Open a terminar in the repository (only dev mode)
```
cd frontend 
ng serve –open
``` 
## Angular deploy
```
cd frontend 
ng build --prod
``` 
Copy the content of /frontend/dist to ASP.NET project root
Add to all local url in index.html the prefix static/<prev_url>
